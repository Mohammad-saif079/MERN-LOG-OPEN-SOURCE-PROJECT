## Folder Structure
### Indusrty Grade Folder Structure
```Bash
server/
├── src/
│   ├── config/              # Database and external service configuration
│   ├── controllers/         # Route business logic
│   ├── middlewares/         # Custom middleware functions
│   ├── models/              # MongoDB schemas and models
│   ├── routes/              # API route definitions
│   └── utils/               # Helper and utility functions
│
├── app.js                   # Express app configuration
├── server.js                # Application entry point
├── .env                     # Environment variables
└── package.json
```
##  To Run locally in your system 
### 1. Clone the Repository
```Bash
git clone https://github.com/ANKITKUMAR1809/MERN-LOG-OPEN-SOURCE-PROJECT.git
```
### 2. Change Dir to MERN-LOG-OPEN-SOURCE-PROJECT/server
```Bash
cd MERN-LOG-OPEN-SOURCE-PROJECT/server
```
### 3. Run Build command
```Bash
npm i
```
### 4. Start Server
```Bash
npm run dev
```
